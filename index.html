A task list app created with Python using the Python GUI package Tkinter. Once opened, the application will display:

  • A selection list housing to-do list items.
  • An “Add Task” button to add a task to the list.
  • A “Remove Task” button which deletes the selected task from the list.
  • An input box for the user to type the task they want to add to the list.
